Estrutura de um sistema operativo

Elementos constituintes:

  • Kernel;

    • O kernel basicamente é o cérebro do sistema operativo, pois estabelece a ligação entre o processamento de dados e os programas.

  • Rede;

    • Esta parte também é extremamente importante, pois permite ao utilizador estabelecer ligações com outros computadores, servidores e outros dispositivos.

  • Segurança;

    • Todos os sistemas operativos contêm aplicações que asseguram a segurança do mesmo, como por exemplo a firewall do Windows.

  • Shell;

    • O que a shell faz basicamente é estabelecer uma comunicação direta entre o usuário e o sistema operativo que este opera.
